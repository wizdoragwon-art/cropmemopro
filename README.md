# Crop Memo Pro — 오프라인 현장 수집 PWA

폰·패드에서 **인터넷 없이 야장 데이터를 저장**하고, 연결되면 **Google Sheets로 동기화**하는 설치형 웹앱(PWA)입니다. 앱스토어·APK 없이 링크만으로 "홈 화면에 추가"해서 씁니다.

## 폴더 구성
```
index.html            앱 화면(셸)
app.js                로직: 오프라인 저장(IndexedDB)·수집·선발·조사일·동기화·CSV
styles.css            디자인 토큰·컴포넌트
icons.js              오프라인 아이콘(인라인 SVG)
xlsx.min.js           엑셀(.xlsx) 파서(SheetJS · 오프라인)
ocr/                  라벨 OCR 엔진(Tesseract · 영문 데이터 · 오프라인 · 약 8.6MB)
manifest.webmanifest  설치 정보(이름·아이콘)
sw.js                 서비스워커(오프라인 캐시)
icons/                앱 아이콘(192·512·maskable·apple)
```

## 1) 실행/호스팅
서비스워커는 `file://`에서는 동작하지 않습니다. **로컬 테스트**는 폴더에서:
```
python3 -m http.server 8000
```
→ 브라우저에서 `http://localhost:8000` 접속.

**팀에 배포**하려면 정적 호스팅에 폴더를 올려 URL을 만드세요(무료). 예: GitHub Pages
1. GitHub에 저장소 생성 → 이 폴더의 파일들을 업로드
2. Settings → Pages → Branch: main / (root) 저장
3. 몇 분 뒤 나오는 `https://<계정>.github.io/<저장소>/` 주소를 폰·패드에서 열기

(Netlify Drop, Vercel, Cloudflare Pages, Google Sites 등 어떤 정적 호스팅도 가능합니다.)

## 2) 설치 (홈 화면에 추가)
- **Android(크롬)**: 주소 열기 → 메뉴(⋮) → "앱 설치" 또는 "홈 화면에 추가"
- **iPad/iPhone(사파리)**: 공유 버튼 → "홈 화면에 추가"

설치하면 앱 아이콘으로 실행되고 전체화면으로 뜹니다.

## 3) 오프라인
첫 로딩 이후에는 인터넷이 없어도 실행됩니다. 입력값은 즉시 기기 안(IndexedDB)에 저장되고, 앱을 껐다 켜도 유지됩니다. 상단 홈의 동기화 카드가 **오프라인 · 미전송 N건**으로 상태를 보여줍니다.

## 4) 동기화 연결 (선택)
사무실 시트로 올리려면 함께 제공된 **`CropMemoPro_GAS_Code.gs`** 를 배포하세요.
1. `sheets.new` → 확장 프로그램 → Apps Script → 코드 붙여넣기 → `setup` 1회 실행
2. 배포 → 새 배포 → 웹 앱 → 액세스 "모든 사용자" → **`/exec` URL** 복사
3. 앱의 **설정 탭 → 동기화 URL**에 붙여넣고 저장 → "연결 테스트"로 확인
4. 이후 온라인이면 자동 전송되고, 홈/내보내기에서 "지금 동기화"로 즉시 전송

> 참고: Apps Script 웹앱은 브라우저 CORS 정책상 배포 액세스가 "모든 사용자"여야 응답을 읽습니다. 실패해도 데이터는 로컬에 안전하게 남아 다음 연결 때 다시 전송됩니다.

## 5) 내보내기
설정과 무관하게 **내보내기 탭 → CSV 내보내기**로 언제든 다운로드됩니다. UTF-8 BOM 롱포맷(엑셀 호환)이며 열은: 라벨번호·세대·반복·개체·**조사일**·형질·값·**개체선발**·**계통선발**.

## 이번 버전(v1) 범위
- 포함: 홈(동기화 상태·이어서 수집 · **과제 목록/새 과제**), **새 과제 생성 마법사**(3단계 · 작물 6종+직접 추가 · 라벨 접두어 · 세대 중복선택 · RCBD 반복 · 형질세트 선택), **과제 수정·삭제**(홈에서 연필/휴지통), **라벨 일괄등록**(과제 수정 → **엑셀 .xlsx** 또는 CSV 업로드로 라벨·반복·개체수 한 번에 채우기 · 오프라인 파서 내장 · 빈 양식 내려받기), **야장 수집**(개체 좌우 스와이프, **형질 수정·추가·삭제**(형질세트 편집), 조사일/시계열, 형질 7종 입력 — 수치·**비율%**·등급·항목·카운터·날짜·문자, 개체/계통 선발 + 필드맵 큰별/작은별·선발만 보기, 계통 평균, 라벨번호 탭 정보수정 · **라벨 OCR 스캔**(카드 '스캔' → 촬영 → 인쇄 라벨 인식 → 교정 → 적용)), **분석 대시보드**(형질별 요약·CV% · 계통 순위 · 일원분산분석 F/p · 유전력 H² · Tukey 평균분리 · 항목형 분포), 내보내기(CSV), 설정(동기화·초기화·**진동 피드백**). 버튼·스와이프 시 짧게 진동합니다(안드로이드 지원, iOS는 미지원). 데모용 토마토 F3 세대(24계통×10개체)가 미리 들어 있습니다.
- 다음 버전 예정: 그리기·사진·음성, 양방향 pull 동기화.

로컬 데이터는 설정 탭의 "초기화"로 지울 수 있습니다.
